=== StockPro v1.0.0 - Installation Windows ===

Option 1 - Version portable (recommandée) :
  1. Décompresser StockPro-Portable.zip
  2. Lancer start.bat
  3. StockPro s'ouvre dans votre navigateur

Option 2 - Installateur complet (nécessite Windows + Inno Setup) :
  - Se placer dans le dossier build/ du projet source
  - Exécuter : python make_installer.py
  - Cela génère StockPro-Setup.exe
 
Données stockées dans %APPDATA%\StockPro\ ou --data-dir.
