=== StockPro v1.3.9 - Installation Windows ===

Installation :
  1. Lancer StockPro-Setup.exe
  2. Suivre les étapes de l'installateur
  3. StockPro s'ouvre automatiquement dans votre navigateur

Aucune dépendance requise. Python est intégré dans l'application.

Données stockées dans %LOCALAPPDATA%\StockPro\ (choisissable à l'installation).

Accès local    : http://127.0.0.1:5001
Accès réseau   : http://<IP_DU_PC>:5001 (ex: http://192.168.1.42:5001)

Pour connaître l'IP du PC : ouvrir cmd et taper  ipconfig
