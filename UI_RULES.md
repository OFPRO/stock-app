# UI RULES

Prioritize:
- operational efficiency
- fast navigation
- dense information display
- readable tables
- keyboard-friendly workflows

Never prioritize aesthetics over productivity.

Use:
- compact spacing
- clear visual hierarchy
- reusable components
- consistent typography

Optimize:
- dashboards
- filters
- forms
- tables
- inventory flows

Target style:
- modern ERP
- Linear inspired
- Stripe dashboard clarity
- SAP Fiori operational usability
